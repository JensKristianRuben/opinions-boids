# Opinion Boids

En interaktiv visualisering af opinions-boids algoritmen, bygget med React og Vite.

## Kom i gang

For at køre projektet lokalt, følg disse trin:

1. **Klon projektet**
   ```bash
   git clone https://github.com/JensKristianRuben/opinions-boids.git
   cd opinion-boids
   ```

2. **Installer afhængigheder**
   ```bash
   npm install
   ```

3. **Start udviklingsserveren**
   ```bash
   npm run dev
   ```

4. **Åbn applikationen**
   Åbn din browser og gå til [http://localhost:5173](http://localhost:5173)

## Teknologier

- React
- Vite
- Tailwind CSS
- Lucide React (for ikoner)
